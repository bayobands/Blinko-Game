# BINKO (GDevelop Game)

Inspired by the creative gameplay of *Balatro*, **BINKO** puts a fun twist on the classic *Plinko* experience. Players drop a ball into a dynamic grid of pegs, aiming to score big by strategically obtaining and using power-ups.
